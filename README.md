# Adatbázis-kezelés alapjai (MySQL)
## ITMP Klub képzés infotanároknak

Az egynapos, gyakorlatorientált képzés a MySQL alapok két részes tanári képzésünk első része, amely során **Rostagni Csaba** vezető mentor segítségével lépésről lépésre ismerkedhetsz meg az SQL lekérdezések felépítésével. Egyszerűbb és összetettebb lekérdezéseket készítesz, kipróbálod a leggyakrabban használt SQL függvényeket, és magabiztos alapokat szerzel a továbblépéshez.

A képzés elsősorban Digitális kultúrát tanító kollégáknak szól, de a szakképzésben Adatbázis-kezelést oktatók is bátran csatlakozhatnak.
